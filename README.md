### supported audio formats:

- `MP3`
- `AAC`
- `WAV`
- `FLAC`
- `OGG`
- `WMA`
- `AIFF`
- `ALAC`

### supported video formats:

- `MP4`
- `AVI`
- `MKV`
- `MOV`
- `WMV`
- `FLV`
- `WebM`
- `H.264`

### supported image formats:

- `JPEG`
- `PNG`
- `GIF`
- `BMP`
- `TIFF`
- `WebP`
- `HEIF`
- `JPEG 2000`
